A production-grade AI health research companion that understands your medical query, searches PubMed + OpenAlex + ClinicalTrials.gov simultaneously, ranks 400+ papers semantically, and delivers grounded, hallucination-free research summaries with local clinical trials.

🎯 Problem: Patients can't find relevant medical research across thousands of papers.

💡 Solution: A pipeline that understands intent → ranks by evidence → verifies facts → personalizes by location.

<details> <summary><b>🔍 See all features</b></summary>
🧠 Semantic Ranking — TF-IDF cosine similarity scores papers against clinical intent
🔬 Subgroup Tagging — Auto-tags ALK+, EGFR Exon 20, NSCLC, SCLC — prevents dangerous generalizations
📍 Location-Aware Trials — City → Country → Global fallback — Indian user gets Indian trials with contacts
🛡️ Anti-Hallucination — 6-layer pipeline verifies every HR, OS, PFS number against paper abstract
💬 Context Memory — Preserves disease across turns — "Can I take Vitamin D?" uses lung cancer context
📊 Evidence Diversity — Max 3 per type — mix of Phase 3 RCTs, Meta-analyses, Phase 2
🎙️ Voice Input — OpenAI Whisper transcription
📥 Export — PDF + Text with citations
</details>
Stack: React Node.js Express MongoDB Redis LLaMA 3.1 8B Groq TF-IDF PubMed API OpenAlex API ClinicalTrials.gov API
